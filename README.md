# IntelEval

**Explanation-Driven Code Evaluation Platform**

IntelEval is a web-based evaluation platform designed to assess coding ability
across **two equally important dimensions**:  
**code quality** and **clarity of explanation**.

Instead of evaluating solutions only by correctness, IntelEval encourages users
to explain *why* their solution works. These two scores are combined into a
single unified metric called the **Intel Eval Index**, providing a more complete
measure of understanding and problem-solving ability.
